🧮 Calculator Web App

A simple and responsive calculator built using HTML, CSS, and JavaScript.
This project performs basic arithmetic operations and demonstrates core frontend development skills.

🚀 Features

➕ Addition

➖ Subtraction

✖️ Multiplication

➗ Division
